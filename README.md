# Destination Paradise 🌍

A full-stack travel booking web application that enables users to explore destinations, create accounts, and book trips through a secure and user-friendly interface.

## Features

- User registration and login with Firebase Authentication
- Browse and explore travel destinations
- Secure trip booking system
- User dashboard for managing bookings
- Responsive and intuitive user interface
- RESTful backend using Node.js and Express.js

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, EJS
- **Backend:** Node.js, Express.js
- **Database:** Firebase Firestore
- **Authentication:** Firebase Authentication

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Destination-Paradise.git
```

2. Navigate to the project

```bash
cd Destination-Paradise
```

3. Install dependencies

```bash
npm install
```

4. Configure Firebase credentials.

5. Start the server

```bash
npm start
```

6. Open your browser and visit:

```
http://localhost:3000
```
